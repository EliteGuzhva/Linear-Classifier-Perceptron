ЛР1. Линейный классификатор и однослойный перцептрон
====================================================

## Запуск
> ```python3 main.py```

## Конфигурация
В файле main.py есть два поля: *DATASET* и *CLASSIFIER*.

> **DATASET:**
> - lego_figures
> - chinese_mnist

> **CLASSIFIER:**
> - linear_classifier
> - perceptron
